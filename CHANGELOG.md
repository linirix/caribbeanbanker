# Changelog

## 1.0.0 - 2026-04-19

Initial public release candidate.

Highlights:
- terminal-based central banking strategy game with historical and randomized campaigns
- short and extended timelines
- historical and teaching scenarios with bespoke objectives
- policy, cabinet, communication, and crisis-decision systems
- reporting tools including `history`, `news`, `report`, `why`, and `tutorial`
- data-driven tuning and scenario configuration
- headless balance harness for seeded automated playthroughs
- packaged macOS release workflow with bundled resources and editable config
